# 📊 Relatório de Vendas e Lucros com Data Analytics no Power BI

## 🎯 Objetivo do Projeto

Este projeto foi desenvolvido durante a Formação **Power BI Analyst (DIO)**, no módulo **Data Analytics & Storytelling com Power BI**.

O desafio teve como objetivo construir um relatório analítico focado em vendas e lucratividade, utilizando recursos de Data Analytics, visualização de dados e DAX para apoiar a tomada de decisões gerenciais.

Além da criação dos visuais, o projeto também teve foco em experiência do usuário (UX), storytelling e organização analítica das informações.

---

# 🧠 Conceitos Aplicados

Durante o desenvolvimento do relatório, foram aplicados conceitos fundamentais de Business Intelligence e análise de dados:

* Data Analytics
* Storytelling com dados
* Experiência do usuário (UX)
* Análise exploratória
* Segmentação e compartimentação de dados
* Criação de medidas em DAX
* Organização visual para tomada de decisão
* Análise de lucratividade
* Visualização estratégica de indicadores

---

# 📚 Estrutura do Relatório

O relatório foi desenvolvido com foco em clareza visual, análise gerencial e navegação intuitiva, permitindo identificar rapidamente padrões e oportunidades dentro dos dados.

O projeto contempla páginas analíticas e detalhamento das informações, organizadas para facilitar a interpretação dos resultados.

---

# 📈 Visuais Desenvolvidos

## 🔹 TOP 3 Produtos

Foram criados visuais para destacar os produtos com maior desempenho em vendas e lucratividade.

Essa análise ajuda a identificar:

* produtos mais relevantes
* impacto no faturamento
* oportunidades estratégicas

---

## 🌎 Principais Países por Vendas e Profit

Desenvolvimento de análises comparativas entre países considerando:

* vendas
* lucro (profit)
* desempenho financeiro

O objetivo foi facilitar a identificação de mercados com maior impacto nos resultados.

---

## 📉 Gráfico de Dispersão

Foi utilizado gráfico de dispersão para analisar a relação entre:

* unidades vendidas
* vendas por mês

Esse tipo de visual permite identificar padrões, concentração de dados e possíveis comportamentos fora da curva.

---

## 🧩 Agrupamentos e Compartimentação

O relatório também inclui técnicas de:

* agrupamento de dados
* segmentação analítica
* compartimentação das informações

Esses recursos ajudam a tornar a análise mais organizada e intuitiva para o usuário final.

# ⚡ Medidas DAX Utilizadas

Durante o projeto, foram criadas medidas em DAX para suportar as análises e indicadores do relatório.

As medidas foram utilizadas para:

* cálculos de vendas
* lucro
* comparações
* análises agregadas
* indicadores estratégicos

👉 Aqui está o ponto-chave:

O DAX permite transformar dados brutos em métricas analíticas relevantes para o negócio.
